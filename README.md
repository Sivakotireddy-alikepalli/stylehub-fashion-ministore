# 🛍️ StyleHub Fashion Mini Store

A full-stack e-commerce platform built with modern technologies.

## 🚀 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy
- SQLite / MySQL

### Features
- User Authentication (Login/Register)
- Product Listing
- Cart System
- Checkout & Stripe Payment
- Order History
- Admin Dashboard
- Coupons System
- Email Notifications (Invoice)

## 🧑‍💻 How to Run

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8001
